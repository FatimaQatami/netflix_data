# Netflix Data Analysis

Exploratory data analysis (EDA) on the Netflix titles dataset using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset 
Netflix movies and TV shows data including title, type, release year, rating, and more.

## What's Inside
- Data cleaning
- Exploratory analysis
- Visualizations

Dataset retrieved from:
https://www.kaggle.com/datasets/shivamb/netflix-shows
